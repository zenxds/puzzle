# 5 岁儿童拼图游戏 — 设计文档

## 目标

为 5 岁小朋友设计的浏览器拼图游戏，主题贴近孩子喜爱的小马宝莉 / 汪汪队 / 卡皮巴拉等动画风格。强调"零挫败、强反馈"，让孩子从拼图过程中获得成就感。

## 关卡设计

5 关递进难度，每关一张主题图：

| 关卡 | 切块网格 | 碎片数 | 主题 |
|------|----------|--------|------|
| 1 | 4×4 | 16 | 彩虹小马（小马宝莉风） |
| 2 | 4×5 | 20 | 救援小狗（汪汪队风） |
| 3 | 5×5 | 25 | 卡皮巴拉泡温泉 |
| 4 | 5×6 | 30 | 独角兽与彩虹城堡 |
| 5 | 6×6 | 36 | 动物大派对 |

## 玩法

- **拖拽放置**：左侧打散的碎片区，孩子用鼠标或手指拖拽碎片到右侧拼图板对应格子
- **吸附辅助**：当碎片距离正确位置 < 25px 时自动吸附 + 播放"叮"声
- **错放回弹**：拖到错位置会平滑弹回原碎片区，不显示红色或"错误"提示（避免挫败）
- **原图预览**：右上角"看原图"按钮，按住时半透明叠加显示完整原图作为参考
- **关卡完成**：所有碎片归位后，触发撒花/星星动画 + 完成乐句 + "你真棒！"鼓励文字
- **下一关**：完成后自动解锁下一关，按"下一关"按钮进入

## 关卡选择界面

- 进入游戏先看到关卡选择网格（5 张卡片）
- 卡片展示主题缩略图 + 关卡编号
- 已通关的关卡显示 ⭐
- 当前可玩的最高关卡正常显示，更高关卡显示锁🔒图标
- 通关进度存 `localStorage`

## UI / 视觉风格

- **目标受众**：5 岁，识字有限，操作以拖拽为主
- **配色**：糖果色（粉 #FFB6D9 / 紫 #C8A8E9 / 天蓝 #A0E1FF / 奶黄 #FFE5A0）
- **按钮**：大号、圆角、明显阴影，hover 有放大动效
- **字体**：圆润字体，16px 起步，关键文字 24px+
- **响应式**：桌面 / 平板 / 手机自适应（孩子常用 iPad）

## 技术方案

### 技术栈
- **零依赖**：纯 HTML5 + CSS3 + 原生 JavaScript（ES2020+）
- **零构建**：双击 `index.html` 直接在浏览器运行
- **可选 yarn**：仅用于本地起静态服务器（不强制）

### 关键实现点

**拼图碎片渲染**
- 用 `<div>` + `background-image` + `background-position` + `background-size` 切割原图
- 每个碎片 = 一个 div，大小 = 拼图板宽度 / 列数
- 形状为方形（5 岁孩子识别方形最容易，逻辑也最简单）
- 碎片初始位置：在左侧"碎片区"随机布局

**拖拽**
- 使用 Pointer Events（`pointerdown` / `pointermove` / `pointerup`），同时支持鼠标和触屏
- 拖拽时碎片 `position: fixed`，跟随指针
- 释放时计算与目标格子的距离，< 25px → 吸附；否则 → 平滑动画回原位

**吸附判定**
- 拼图板的每个目标格子有固定坐标
- 距离比较使用碎片中心点

**音效（Web Audio API 合成，零外部文件）**
- "叮"声：高频正弦波 + 快速衰减
- 完成乐句：C-E-G-C 上行三和弦
- 背景音乐：4 小节循环的简单童谣旋律（可静音）
- 首次用户交互后才创建 AudioContext（绕过浏览器自动播放限制）

**完成庆祝**
- Canvas 粒子动画：撒花 + 星星从中心爆开，10 秒淡出
- DOM 文字弹出："你真棒！"+ 缩放进入动画

**进度存储**
- `localStorage` key: `puzzle.progress` = `{ completedLevels: number[], highestUnlocked: number }`

## 测试方案

人工浏览器测试（无单元测试，因为这是一个交互密集的小游戏，UI 行为最重要）：

1. 关卡 1 → 通关 → 关卡 2 解锁
2. 刷新页面 → 通关进度保留
3. 拖拽碎片到正确位置 → 吸附 + 叮声
4. 拖拽到错误位置 → 弹回
5. 长按"看原图" → 显示原图遮罩
6. 触屏（iPad / 手机）操作正常
7. 5 张图片在 4×4 ~ 6×6 切割下都不变形
